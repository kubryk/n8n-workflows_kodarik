# n8n-workflows_kodarik

Як користуватися?

1. Скачай потрібний .json файл із папки workflows/.
2. У n8n обери Import from File → вибери JSON.
3. Додай свої API ключі / credentials, якщо вони потрібні.
4. Активуй workflow.
